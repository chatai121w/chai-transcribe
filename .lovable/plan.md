## מטרה
שתי העמודות בעורך (`SyncMirrorLayout`) זהות 1:1 ועריכות הדדית — שינוי בצד אחד מתעדכן מיידית בשני. סרגל צף מחולק לשני חצאים. אפשרות לנעול צד אחד שלא ישתנה. צבע אייקונים שונה רק על הצד הפעיל/הנבחר.

## מה משתנה

### 1. שתי עמודות זהות — מירור מלא 1:1
- העמודה הימנית (כיום read-only mirror) תוחלף ב-`RichTextEditor` שני, זהה לחלוטין לשמאלי.
- שתיהן משתמשות באותו `text` ובאותו `onTextChange` → single source of truth.
- שינוי מילה / פיסוק / פסקה באחת — מופיע מיידית בשנייה (אותו state, אותו רינדור).
- הסגנון (גופן, גודל, צבע, יישור, גובה שורה) משותף בין שתיהן — אין סגנון נפרד לעמודה.

### 2. בחירת עמודה פעילה + נעילה
- State חדש: `activePane: 'right' | 'left'` — איזו עמודה נחשבת "הפעילה" כרגע.
- State חדש: `lockedPane: 'right' | 'left' | null` — איזו עמודה נעולה (read-only).
- כל עמודה תקבל לחצן קטן בכותרת:
  - **לחצן בחירה** (סמן/עיגול) — מסמן אותה כפעילה.
  - **לחצן מנעול** (Lock/Unlock icon) — נועל את הצד הזה. נעילה הופכת אותו ל-readOnly ולא מאפשרת onChange.
- אם מנסים לערוך צד נעול → toast: "צד זה נעול. שחרר את הנעילה כדי לערוך".
- ויזואלית: צד נעול מקבל אטימות 0.85 ורקע מעט אפור.

### 3. סרגל צף מחולק לשני חצאים
- הסרגל הצף הקיים (`sticky top-0`) יוצג כ-flex עם שני בלוקים + מחיצה (`border-l border-border`) באמצע.
- חצי ימני: כפתורי סגנון/פעולות המתייחסים לעמודה הימנית (גם כשהפעולות גלובליות, הצד הימני מציין שזה צד ימין).
- חצי שמאלי: זהה, לצד שמאל.
- כפתורי שמירה / בסיס / השוואה / חזרה לבסיס — נשארים כפעולות גלובליות בקצה (לא משתייכים לחצי מסוים).

### 4. צבע אייקונים לפי עמודה פעילה
- **עמודה פעילה (נבחרה)**: אייקונים בכחול נייבי עמוק `text-[#0a1d3f]`.
- **עמודה לא-פעילה**: אייקונים באפור בהיר `text-muted-foreground`.
- המעבר מתבצע מיידית בלחיצה על "סמן כפעיל" באחת העמודות.
- צד נעול: האייקונים מקבלים אופסיטי 0.5 בנוסף.

## פרטים טכניים

**`SyncMirrorLayout.tsx`:**
- החלפת בלוק הרינדור של העמודה הימנית (סביב שורה ~1281) ב-`RichTextEditor` נוסף עם אותו `value={text}` ו-`onChange={(v) => lockedPane !== 'right' && onTextChange(v)}`.
- ה-`TextMarkingOverlay` ישוכפל לשתי העמודות (אותה לוגיקה של highlight של מילה פעילה).
- הסגנון המשותף (`textStyle`) נשאר אחד — חל על שתיהן.
- ה-floating toolbar (כבר קיים) יקבל wrapper `flex` עם `<div className="flex-1">{right half}</div><div className="border-l" /><div className="flex-1">{left half}</div><div>{global actions}</div>`.
- כל `<Icon className="..." />` בתוך הסרגל יקבל class דינמי: `activePane === 'right' ? 'text-[#0a1d3f]' : 'text-muted-foreground'` (ולהפך לצד שמאל).
- כפתורי בחירה ונעילה: יתווספו בפינה הימנית/שמאלית של כל עמודה (`Circle`/`CheckCircle` ל-בחירה, `Lock`/`Unlock` לנעילה).

**אין שינוי ב-business logic, baseline, או שמירה.**

## אימות
- הקלדה בצד ימין → מופיעה מיידית בצד שמאל ולהפך.
- לחיצה על "נעל" בצד ימין → אי-אפשר להקליד בו, צד שמאל ממשיך לעבוד.
- בחירת צד פעיל → רק האייקונים שלו בכחול נייבי, השני באפור.
- יישור שורה-מול-שורה נשמר במלואו.
