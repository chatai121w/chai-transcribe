## 💾 3 מצבי השמירה

### ☁️ מצב 1: שמור לענן (ברירת מחדל)

```
                    ✅ תמלול הסתיים
                          │
           ┌──────────────┴──────────────┐
           ▼                             ▼
   📝 הטקסט המלא                 🔊 קובץ האודיו
   נשמר ב-Supabase               נשמר ב-Supabase
           │                             │
           └──────────────┬──────────────┘
                          ▼
                ✅ מופיע בהיסטוריה
                ✅ נגיש מכל מכשיר
                ✅ גיבוי בענן
                ⏱️ לוקח 2-10 שניות נוספות
                   (תלוי בגודל הקובץ)
```

**מתי להשתמש:**
- ✅ רוצה גיבוי מלא לכל מה שמתמללת
- ✅ עובד ממכשירים שונים
- ✅ צריך לגשת לתמלולים ישנים

**מה קורה מאחורי הקלעים:**
```python
# שמירה מלאה — טקסט + אודיו
saveToHistory(text, engine)        # טקסט → Supabase DB
uploadAudioFile(file)              # אודיו → Supabase Storage
```

### 📝 מצב 2: טקסט בלבד

```
                    ✅ תמלול הסתיים
                          │
                          ▼
                   📝 הטקסט המלא
                   נשמר ב-Supabase
                          │
                          ▼
                ✅ מופיע בהיסטוריה
                ✅ נגיש מכל מכשיר
                ❌ קובץ האודיו לא נשמר
                ⚡ מהיר יותר (~0.5 שניות)
```

**מתי להשתמש:**
- ✅ רוצה היסטוריית תמלולים אבל לא צריך את קובץ הקול
- ✅ חוסך מקום בענן
- ✅ הקובץ גדול ולא רוצה להמתין להעלאה

**מה קורה מאחורי הקלעים:**
```python
# שמירת טקסט בלבד
saveTextOnlyToCloud(text, engine)  # טקסט → Supabase DB
# הקובץ לא עולה!
```

### 🏠 מצב 3: מקומי בלבד (הכי מהיר!)

```
                    ✅ תמלול הסתיים
                          │
                          ▼
                   💾 localStorage
                   (זיכרון הדפדפן)
                          │
                          ▼
                ✅ זמין מיידית
                ⚡ הכי מהיר — אין המתנה
                ❌ לא נגיש ממכשיר אחר
                ❌ אם תנקה דפדפן — יימחק
                ❌ לא מופיע בהיסטוריה הענן
```

**מתי להשתמש:**
- ✅ עובדת מהיר ואין זמן להמתין
- ✅ תמלול לצורך חד-פעמי
- ✅ פרטיות מוחלטת — שום דבר לא יוצא מהמחשב

**מה קורה מאחורי הקלעים:**
```javascript
// שמירה מקומית בלבד
localStorage.setItem('transcript_history', JSON.stringify(entry));
// שום קריאה לשרת הענן!
```

## 🔧 הגדרות טכניות (למי שרוצה לדעת יותר)

### איפה שמור ההגדרה?

```
localStorage key: 'cuda_cloud_save'
ערכים אפשריים:   'immediate' | 'text-only' | 'skip'
ברירת מחדל:      'immediate'
```

### איך הקוד מחליט מה לשמור?

```javascript
const cloudSaveMode = preferences.cuda_cloud_save || 'immediate';

if (cloudSaveMode === 'skip') {
  // שומר רק ב-localStorage — אין קריאות ענן
  saveToHistory(text, engine, skipCloud: true);

} else if (cloudSaveMode === 'text-only') {
  // שומר טקסט בלבד ל-Supabase
  saveTextOnlyToCloud(text, engine);

} else {
  // ברירת מחדל: שומר הכל — טקסט + אודיו
  saveToHistory(text, engine);
}
```
