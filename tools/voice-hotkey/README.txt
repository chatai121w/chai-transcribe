🎙️  קלט קולי עברית — Whisper
==============================

כלי קלט קולי גלובלי לעברית:
לחץ קיצור מכל מקום ב-Windows → דבר → הטקסט מודבק אוטומטית.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 📁 תוכן התיקייה
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  voice_hotkey.py       — הסקריפט הראשי (Python)
  start.bat             — הפעלה ידנית עם חלון קונסול
  start-hidden.vbs      — הפעלה ללא חלון (לשימוש הפעלה אוטומטית)
  install-startup.bat   — 🔧 התקנה: הוסף הפעלה אוטומטית עם Windows
  uninstall-startup.bat — 🗑️  הסרה: בטל הפעלה אוטומטית
  README.txt            — קובץ זה

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 🚀 הפעלה ראשונה
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. ודא ששרת Whisper פעיל (port 3000)
2. לחץ פעמיים על start.bat
   — חלון קונסול יפתח ויראה שהכלי פעיל

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 ⚙️  התקנה — עלייה עם Windows
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

לחץ פעמיים על:   install-startup.bat

הכלי יתחיל אוטומטית עם כל כניסה ל-Windows (ברקע, ללא חלון).

להסרה:           uninstall-startup.bat

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 🎹 שימוש
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

 CTRL+SHIFT+H  →  התחל הקלטה
                   (בר אדום מופיע בתחתית עם מד עוצמה)
 CTRL+SHIFT+H  →  עצור + תמלל
                   (Whisper מעבד → הטקסט מודבק אוטומטית)
 Escape        →  בטל הקלטה

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 🔧 שינוי הגדרות
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

לשינוי קיצור המקשים, ערוך את start.bat:
  "%PYTHON%" "%HERE%voice_hotkey.py" --hotkey ctrl+alt+h

אפשרויות:
  --hotkey ctrl+shift+h   קיצור מקשים (ברירת מחדל)
  --port 3000             פורט שרת Whisper
  --lang he               שפה (he=עברית, en=אנגלית, yi=יידיש)
  --beam 3                דיוק מול מהירות (1=מהיר, 5=מדויק)

 📋 דרישות
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  • Python 3.10+ עם venv בשורש הפרויקט (.venv)
  • pip install sounddevice numpy requests
  • שרת Whisper פעיל (server/transcribe_server.py)
  • Windows בלבד (משתמש ב-Windows API לקיצורי מקשים גלובליים)
